# 🗂️ Kanban Board Task Manager

A simple and interactive **Kanban Board** web application that helps users manage tasks efficiently using a drag-and-drop interface. Tasks are categorized into **To Do**, **In Progress**, and **Done**, with persistent storage using LocalStorage.

---

# Features

*  Add new tasks with title and description
*  Drag & drop tasks between columns
* Delete tasks instantly
*  Data persistence using LocalStorage (tasks remain after refresh)
*  Dynamic task count per column
*  Clean and minimal UI

# How It Works

* Tasks are created dynamically using JavaScript
* Drag-and-drop functionality is implemented using native HTML5 Drag Events
* All tasks are stored in the browser using **LocalStorage**
* On page reload, tasks are fetched and rendered automatically
